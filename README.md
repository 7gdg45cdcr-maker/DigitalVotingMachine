# Digital Voting Machine with Secure Memory

## Features
- Vote counting using FSM (Finite State Machine)
- EEPROM based secure vote storage
- Debounce logic for push button voting
- 7 Segment / LCD display output
- Verilog HDL implementation
- GTKWave simulation verification

## Files
- voting_fsm.v → FSM logic
- debounce.v → switch debouncing
- eeprom_memory.v → secure storage
- display_controller.v → display logic
- seven_seg_decoder.v → display decoder
- top_module.v → top integration
- tb_top_module.v → testbench

## Tools Used
- Verilog
- Icarus Verilog
- GTKWave
- VS Code
- GitHub
